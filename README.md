# Ejecutar en Desarrollo

1. Clonar Repo
```
git clone 
```
2. Ejecutar

```
  yarn install
```
3. Tener el Nest CLI instalado
```
  npm i -g @nest/cli
```
4. Levantar la base de datos 
```
docker-compose up -d
```
## Stack Usado
 * Mongo DB
 * Nest
